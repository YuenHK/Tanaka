## 什麼是Tanaka?

[Tanaka](https://yuenhk.github.io/Tanaka)(日文音譯：田中)是一個維修記錄工具，用於回報和管理組織內一切維修項目。

## 功能

Tanaka分為使用者和管理員兩種權限，功能上各有不同。

```markdown

# 使用者
- 回報新事項
- 查看自己的回報事項

## 管理員
- 回報新事項
- 查看全部回報事項
- 修改及刪除回報事項
- 修改及刪除維修項目
- 修改及刪除建築地點
- 修改及刪除管理員
- 匯出Excel

```
